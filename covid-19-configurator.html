COVID-19 Configurator

Ground size
The active area of the simulation in meters. Buildings and people will be bound to a square of this size. Range from 1 to 10000 meters (10km), default is 500 meters.
Earth size
The area surrounding the simulation area. Its edge makes the horizon. Range from 1000 to 100000 meters (100km), default is 50000 meters (50km).
Agent count
The maximal number of people in the simulation. Their number might be smaller if the amount of houses and apartment buildings are not sufficient for all of them. Range from 0 to 50000, default is 100 people.
Line formation
If set, the people are positioned in lines and columns. If not set, the people are positioned in buildings. Default is not set.
Auto rotate
If set, the scnese rotates continuously. The user can still use the mouse to rotate manually. If not set, the scene does not move by itself. Default is not set.
Auto rotate speed
The speed of automatic rotation measure in radians per second. The value is effective only if "Auto rotate" is set, otherwise it has no effect. Range from 0.0 to 1.0, default is 0.3 rad/s.
